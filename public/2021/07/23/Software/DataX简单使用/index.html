DataX简单使用
2021-07-23|Software
Word count: 803|Reading time: 3 min

DataX 是由阿里巴巴开源的离线数据同步框架，它支持的数据源是采用插件式管理，目前支持大多数关系型数据库以及非关系型数据库，查看 DataX 支持的数据源可以在 Github 或者 CODE China 查看，也可以对源码进行二次开发。

常用命令

常用的命令只有一个 data.py，使用它有两个目的，一个是查看指定的 reader 和 writer 的模板配置，它采用的是 json 形式的配置方式，简单易理解；另外一个就是根据配置信息启动一个离线同步任务。

每个 reader 或者 writer 包含的依赖都在$DATAX_HOM/plugin目录下，在依赖版本落后时可以替换依赖来完成需求。每次执行任务的 log 可以在$DATAX_HOM/log目录下找到，它是按日期分到不同的文件夹，再按 json 配置文件名加时间来区分不同任务的。

-p 参数特别有用，在使用定时任务的时候就可以在 shell 中给命令传入不同的参数同步指定的数据。
